Technologien

HTML

HTML (Hypertext Markup Language) ist die Standard-Auszeichnungssprache zum Erstellen von Webseiten und Webanwendungen. Es definiert die Struktur eines Dokuments und beschreibt dessen Inhalt.

CSS

CSS (Cascading Style Sheets) wird verwendet, um das Layout und Design von Webseiten zu gestalten. Es ermöglicht die Anpassung von Farben, Schriftarten, Abständen und mehr

JavaScript

JavaScript ist eine Programmiersprache, die verwendet wird, um Webseiten interaktiv zu gestalten. Sie ermöglicht die Erstellung von dynamischen Inhalten, Animationen und die Kommunikation mit Servern.
